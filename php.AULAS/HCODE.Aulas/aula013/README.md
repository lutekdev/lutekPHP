# ESCOPO DE VARIÁVEL 

- Nessa aula foi lecionado sobre os escopos de variável o que mais me chamou atenção é que ao criar uma função, não se pode chamar a variavel que está fora.
  
  - Basicamente você tem que setar um `global $variavel` dentro da função para a função conseguir enxergar a variavel, isso é extremamente util para os desenvolvimentos, pois o que mais utilizamos e funções.
